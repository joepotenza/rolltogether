# Roll Together - Find your group. Start your next campaign.

Roll Together is a site for D&amp;D players and other tabletop gamers who are looking for groups to join or who want to fill seats at their table. The site is designed with Google Calendar integration to make scheduling sessions easier and will eventually have collaborative features allowing groups to customize their gaming experience together.

## Live preview

The live preview can be found at [https://rolltogether.online](https://rolltogether.online)

## Technologies Used

- React.js with Vite
- JSON Web Tokens
- Google OAuth2 with Google Calendar
- Avatar APIs from [DiceBear](https://www.dicebear.com/) and [MultiAvatar](https://multiavatar.com/)
- [Obscenity filter](https://github.com/jo3-l/obscenity)
- [Quill WYSIWYG editor](https://github.com/VaguelySerious/react-quill)
- [React Date Picker](https://github.com/Hacker0x01/react-datepicker)
- Backend developed with Node.js and MongoDB [Click here to view the backend code](https://github.com/joepotenza/rolltogether-backend/pull/1)
